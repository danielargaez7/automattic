# Reference Image Library

Drop royalty-free inspiration images here. The AI will automatically pull the best match based on the user's vibe + site type selection.

## Folder Structure

Each folder = a vibe. Name files: `{vibe}-{siteType}.jpg` for specific combos, or `{vibe}-1.jpg`, `{vibe}-2.jpg` for general vibe references.

```
library/
  minimalist/
    minimalist-1.jpg     ← clean white layouts, lots of whitespace
    minimalist-2.jpg
    minimalist-blog.jpg
    minimalist-portfolio.jpg

  bold/
    bold-1.jpg           ← strong colors, heavy typography, high contrast
    bold-business.jpg
    bold-ecommerce.jpg

  elegant/
    elegant-1.jpg        ← refined, serif fonts, gold/cream tones
    elegant-portfolio.jpg
    elegant-personal.jpg

  playful/
    playful-1.jpg        ← bright colors, rounded shapes, fun layouts
    playful-blog.jpg

  corporate/
    corporate-1.jpg      ← professional, blue tones, structured
    corporate-business.jpg
    corporate-agency.jpg

  organic/
    organic-1.jpg        ← natural textures, greens, earthy tones
    organic-blog.jpg
    organic-business.jpg

  dark/
    dark-1.jpg           ← dark backgrounds, neon accents, moody
    dark-portfolio.jpg
    dark-agency.jpg

  warm/
    warm-1.jpg           ← warm oranges, creams, inviting
    warm-blog.jpg
    warm-personal.jpg
```

## Image Guidelines
- Minimum 800×600px, JPG or WebP
- Must be royalty-free (Unsplash, Pexels, Pixabay, or AI-generated)
- Should show a full website layout or strong visual style reference
- Rich colors are better than plain/generic screenshots
